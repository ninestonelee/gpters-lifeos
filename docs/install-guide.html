🆕 신규 설치

신규 · ⌨️ 터미널

1) 스킬 설치 (1분)

▸ Mac / Linux
bash <(curl -fsSL https://raw.githubusercontent.com/ninestonelee/gpters-lifeos/main/install-skill.sh)
▸ Windows (PowerShell — WSL 필요)
curl.exe -fsSL https://raw.githubusercontent.com/ninestonelee/gpters-lifeos/main/install-skill.sh | wsl bash

WSL이 없다면 아래 Windows WSL 설치 참고.

2) 설치 후 첫 세팅

  1. Claude Code 재시작 (또는 /reload-skills)
  2. /lifeOS-init — 내 볼트 생성 (~/lifeOS_이름/)
  3. /north-star-define — 북극성(가치 3 + 행동 3) 작성
  4. /agent-setup — 나만의 AI 참모 설정 (이름·호칭·톤·역할)
  5. /today · /close-day — 일일 루틴 시작
신규 · 🤖 Claude Code / Codex

1) 에이전트에게 그대로 붙여넣기

터미널을 직접 만지지 않고, AI에게 설치를 맡깁니다. 채팅창에 아래를 붙여넣으세요:

아래 명령으로 인생OS 스킬을 설치해줘:
bash <(curl -fsSL https://raw.githubusercontent.com/ninestonelee/gpters-lifeos/main/install-skill.sh)

Codex CLI도 동일합니다. 에이전트가 명령을 실행하고 결과를 알려줍니다.

2) 이어서 세팅도 부탁

설치됐으면 /lifeOS-init 로 볼트 만들고,
/north-star-define, /agent-setup 까지 차례로 안내해줘.
⚠️ 설치 후 Claude Code 재시작(또는 /reload-skills)을 해야 새 스킬이 인식됩니다.

❓ 자주 묻는 질문

업데이트하면 기존 노트나 설정이 사라지나요?

아니요. 스킬 파일만 추가/최신화하며 볼트의 노트·데이터는 건드리지 않습니다. AI 참모 설정(/agent-setup)도 기존 것을 보존합니다.

"command not found" / "skill not found"가 떠요.

설치 후 Claude Code 재시작 또는 /reload-skills를 실행하세요. 그래도 안 되면 설치 명령을 한 번 더 실행하세요.

Windows인데 WSL이 뭔가요?

아래 WSL 설치를 참고하세요. 리눅스 환경을 윈도우에서 쓰게 해주는 기능입니다.

터미널과 AI 에이전트, 뭐가 더 쉬워요?

처음이라면 AI 에이전트(Claude Code/Codex)에 붙여넣기가 가장 쉽습니다. 명령을 대신 실행하고 다음 단계까지 안내해줍니다.

🪟 Windows WSL 설치

PowerShell을 관리자 권한으로 열고:

wsl --install

재부팅 후 Ubuntu가 설치되면, 위의 신규/업데이트 명령을 그대로 사용하세요.
